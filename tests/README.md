#Usage
If python >= 2.7
    SCORMCloud_PythonLibrary$ python -m unittest discover

Otherwise, you must use the "unit2" script that comes with the unittest2 module
    SCORMCloud_PythonLibrary$ /path/to/unit2 discover

mock
